> `Python` `FastAPI` `InsightFace` `FAISS` `OpenCV` `PostgreSQL` `WebSockets`

A production-grade computer vision system for real-time image and video analytics across multiple simultaneous camera streams.

- **5-stage async pipeline**: RTSP ingestion → InsightFace ArcFace detection → 512-dim FAISS IndexFlatIP vector search → IoU deduplication → PostgreSQL write

### 💰 AI-Driven Dynamic Hotel Pricing System
> `Python` `Apache Kafka` `Prophet` `Gradient Boosting` `FastAPI` `Streamlit` `Docker`

End-to-end ML pricing engine combining real-time streaming, forecasting, and business guardrails.

- **Full pipeline**: Web scraper (retry + synthetic fallback) → Kafka (acks=all, GZIP, 24-hr deque rolling window) → 15+ engineered features → dual-model inference
- **Prophet** (30-day demand forecast) + **Gradient Boosting** elasticity model (5-fold CV, R² > 0.85)
- Stateless `DynamicPricingEngine` with 4-layer guardrails: velocity ±30%, competitor corridor ±15%, floor/ceiling limits
- 6 FastAPI endpoints with Pydantic v2 schemas + background retraining; 4-tab Streamlit dashboard
- Entire stack (Zookeeper, Kafka, API, Dashboard) deployable with a single `docker-compose up`
